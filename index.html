Rulebook

  1. Web-vida is a dynamic website development competition with a time duration of 2 hours.
  2. The event is scheduled for February 17th, 2025, in UG Activity Spaces 1 and 2, at 5:30 PM.
  3. Registration for the event will be done by the team leader through the official Colloquium Website.
  4. The event consists of two rounds. Initially, each team will have 500 points.

Round 1:

  1. A QR code will be displayed initially, redirecting participants to a website.
  2. The website will contain a trivia button, which will lead to a trivia round on web development-related topics.
  3. After completing the trivia round, a leaderboard will be displayed showcasing the points earned.

Round 2:

  1. Teams will then participate to bid their favorable website domain for website building using the points earned in the previous round.
  2. Following, a QR code will be displayed, redirecting to a GitHub repository containing a basic code template.
  3. Teams will be challenged with commands that require them to implement specific features into their website (if completed, extra points will be given).
  4. The Website building round will then continue for 45 minutes, after which the teams are required to submit their source file.
  5. Teams will be judged on creativity, uniqueness and functionality of the website.

5. All participants must bring their laptops fully charged with a stable internet connection.

6. The Organizing Committee reserves the right to disqualify teams found using unfair means, with the decision being final.
